Authentication

All calls that change an asset requires authentication. Authentication is done by adding a user={user_name} parameter to the endpoint. For example, to authenticate to /api/doormen/zST8_FWcPDT2-14wDQAAAQ== with a user called "bob", the url becomes /api/doormen/zST8_FWcPDT2-14wDQAAAQ==?user=bob
